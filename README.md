# Biodata Estetik (Dea Fitri Andina)

Ini adalah biodata personal yang dirancang dengan tampilan estetik menggunakan konsep **Glassmorphism**.

Desain ini menggabungkan warna-warna cerah, animasi slide-in yang lembut, efek glow pada foto profil, dan didukung oleh kerangka kerja Tailwind CSS untuk styling yang cepat dan responsif.
